\hypertarget{purpose}{%
\chapter{Purpose}\label{purpose}}

Welcome to the FlyingGreen User Manual, your comprehensive guide for
navigating and utilising the innovative FlyingGreen platform. This
manual is designed for all users, whether you're a new or an experienced
user, and aims to provide all the necessary information to understand
and operate the FlyingGreen platform effectively.

Within this document, you will find detailed instructions, guidelines,
and visual aids to help you make the most of FlyingGreen's
functionalities.

From data representation to practical implementation, this reference
manual covers every aspect of the FlyingGreen platform, ensuring you
have the knowledge required to leverage its features to their fullest
potential.

Thank you for choosing FlyingGreen.

\hypertarget{flyinggreen-in-brief}{%
\chapter{FlyingGreen in brief}\label{flyinggreen-in-brief}}

Welcome to theFlyingGreenplatform. \textbf{FlyingGreen} platform,
developed by EUROCONTROL's \textbf{AviationSustainability Unit (ASU)} in
cooperation with key aviation stakeholders. As unique one-stop-shop,
FlyingGreen supports ECAC Member States and the aviation sector in their
efforts toward decarbonisation and climate change adaptation. Our
comprehensive toolkit includes the following pillars: •

\begin{itemize}
\item
  \textbf{NetZero:} Embark on a data-driven journey toward achieving
  zero emissions by 2050. Explore what-if scenarios to inform strategic
  decisions.
\item
  \textbf{FuellingDecarb:} Access centralised information on
  transitioning from fossil to sustainable aviation fuels. Utilise our
  `green fuels' calculator and resource hub.
\item
  \textbf{ClimAdapt:} Build resilience to climate change, by using
  ourclimate risk pre-screening or ECO assessment tools and repository
  of information on risk assessment and adaptation.
\item
  \textbf{DecarbFin:} Facilitating access to funding and providing good
  practices and tools for compliance with EU sustainability regulations.
\end{itemize}

\hypertarget{save-retrieve-delete-project-data}{%
\section{Save, Retrieve, Delete Project
data}\label{save-retrieve-delete-project-data}}

Note: You need to log in to enable project data management.

\begin{enumerate}
\def\labelenumi{\arabic{enumi})}
\tightlist
\item
  Display the projects list

  \begin{enumerate}
  \def\labelenumii{\arabic{enumii})}
  \tightlist
  \item
    Open the Project(s) dropdown.
  \item
    The list of available projects appears (e.g., MyFirstProject,
    MySecondProject, New Project).
  \end{enumerate}
\item
  Create a new project and save data

  \begin{enumerate}
  \def\labelenumii{\arabic{enumii})}
  \tightlist
  \item
    In the Project(s) dropdown, choose New Project.
  \item
    In the New Project field, enter the project name (e.g., My New
    Project Name).
  \item
    Review the Net Zero information shown on the page.
  \item
    Click Save to create the project and store the displayed Net Zero
    information.
  \end{enumerate}
\item
  Update data in the current project

  \begin{enumerate}
  \def\labelenumii{\arabic{enumii})}
  \tightlist
  \item
    Ensure the desired project is selected in Project(s) (e.g.,
    MyFirstProject).
  \item
    Make your changes on the page.
  \item
    Click Save to update the current project with the new information.
  \end{enumerate}
\item
  Retrieve project data

  \begin{enumerate}
  \def\labelenumii{\arabic{enumii})}
  \tightlist
  \item
    Open the Project(s) dropdown.
  \item
    Select the project you want to load (e.g., MyFirstProject,
    MySecondProject, or My New Project Name).
  \item
    The stored data for that project will be retrieved and displayed.
  \end{enumerate}
\item
  Delete a project

  \begin{enumerate}
  \def\labelenumii{\arabic{enumii})}
  \tightlist
  \item
    Open the Project(s) dropdown and locate the project to remove.
  \item
    Click the delete (trash/bin) icon next to that project.
  \item
    In the confirmation dialog, click Confirm to permanently delete the
    project (or Cancel to keep it).
  \end{enumerate}
\end{enumerate}

\begin{enumerate}
\def\labelenumi{\arabic{enumi})}
\setcounter{enumi}{1}
\tightlist
\item
  Select the Area / State / FAB / ANSP / Airport

  \begin{enumerate}
  \def\labelenumii{\arabic{enumii})}
  \setcounter{enumii}{1}
  \tightlist
  \item
    Open the Area dropdown.
  \item
    Pick the item that matches your chosen level:

    \begin{enumerate}
    \def\labelenumiii{\arabic{enumiii})}
    \tightlist
    \item
      Network → choose a region (e.g., ECAC, EUROCONTROL, EU27+EFTA,
      EEA, RP3, EU27).
    \item
      State → choose a country (e.g., Belgium).
    \item
      FAB → choose a Functional Airspace Block (e.g., FABEC).
    \item
      Airport → choose an airport (e.g., Brussels).

  \item

  \end{enumerate}

EMAS,GreenATM, ISO 14000 family, etc.\hspace{0pt}

The CANSO GreenATM programme is a voluntary environmental
accreditationdesigned for ANSPs to assess and enhance their
environmental performance. Itoffers a structured framework with five
maturity levels, evaluating areas such asgovernance, operational
efficiency, infrastructure, and stakeholder engagement.\hspace{0pt}

Participation in the GreenATM programme is entirely voluntary for ANSPs.
Itserves as an independent, industry-endorsed accreditation of
theirenvironmental efforts.\hspace{0pt}
